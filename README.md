## Overview
This paper tackles the task of Satellite-to-Street View Synthesis, aiming to generate photorealistic ground-level panoramas and videos from given satellite imagery and camera trajectories. We propose a novel coarse-to-fine two-stage generation framework (Geometry-Aware Coarse Synthesis + PanoDiT) to address the trade-off dilemma between geometric consistency and texture fidelity.
